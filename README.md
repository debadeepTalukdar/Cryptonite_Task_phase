# ML Projects Portfolio — Debadeep Talukdar

End-to-end machine learning projects built with Python, scikit-learn, and pandas.

---

## Projects

### 1. Breast Cancer Classification
- **Goal:** Predict malignant vs benign tumors
- **Dataset:** Wisconsin Breast Cancer Dataset
- **Models Used:** Logistic Regression, Random Forest
- **Result:** ~97% accuracy

### 2. Diabetes Prediction
- **Goal:** Predict diabetes onset from medical features
- **Dataset:** Pima Indians Diabetes Dataset
- **Models Used:** Logistic Regression, KNN
- **Result:** ~78% accuracy

### 3. House Price Prediction
- **Goal:** Predict house prices from features
- **Dataset:** Standard housing dataset
- **Models Used:** Linear Regression, Decision Tree
- **Result:** Low RMSE on test set

### 4. Titanic Survival Prediction
- **Goal:** Predict passenger survival
- **Dataset:** Kaggle Titanic
- **Models Used:** Random Forest, SVM
- **Result:** ~82% accuracy

### 5. Iris Classification
- **Goal:** Classify flower species
- **Dataset:** Iris Dataset
- **Models Used:** KNN, SVM, Decision Tree
- **Result:** ~98% accuracy

### 6. Wine Quality Prediction
- **Goal:** Predict wine quality score
- **Dataset:** UCI Wine Quality
- **Models Used:** Random Forest, XGBoost
- **Result:** ~88% accuracy
